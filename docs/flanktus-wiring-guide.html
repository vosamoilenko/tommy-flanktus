⚡

230V AC is dangerous. Only the brown (live) wire goes through the relay. Blue and yellow/green go straight through Wago connectors. Keep electronics away from water.

03
Breadboard — Button & Sensors
Button D5 (auto toggle)
DS18B20 #1 water (D2)
DS18B20 #2 air (D4)
5V (+) rail
GND (−) rail
ⓘ

Both DS18B20 sensors need a 4.7kΩ pull-up resistor between DATA and VCC. Without it, the sensor reads −127°C. Each sensor is on its own pin (D2 and D4), each with its own pull-up.

04
DS18B20 Waterproof Probe — Wiring Detail
ⓘ

Wire colors may vary by manufacturer. Most waterproof DS18B20 probes use Red/Yellow/Black, but some use Red/White/Black or Red/Blue/Black. If unsure, check with a multimeter: GND and VCC should show ~0Ω, DATA will have high resistance to both.

05
Step-by-Step
Step 1
Connect 3 jumper wires from Arduino to relay:
D7 → IN   5V → VCC   GND → GND
Step 2
Wire push button on breadboard:
D5 → Button → GND
Straddles center channel. No resistor needed (internal pullup).
Step 3
Wire DS18B20 #1 (water temp):
D2 → DATA   5V → VCC   GND → GND
4.7kΩ resistor between DATA and 5V. Probe into reservoir.
Step 4
Wire DS18B20 #2 (air temp):
D4 → DATA   5V → VCC   GND → GND
4.7kΩ resistor between DATA and 5V. Mount outside in shade.
Step 5
Cut the brown wire from the mains cable. One end to relay COM, other to relay NO. Blue + yellow/green go straight through Wago.
Step 6
Install libraries, compile, upload:
arduino-cli lib install "OneWire" "DallasTemperature"
Connect USB, upload sketch. Press button to start auto mode.
